## 🔨 규칙 및 스크럼 작성 방법

[habit-morning Wiki](https://github.com/habit-developers/habit-morning/wiki)를 참고해주세요.

## 🌳 Jayden

- 평일 5일을 기준으로 작성한다.(단, 스크럼은 없어도 작성하고 싶으면 더 작성하자!)
- 큰 디렉토리는 연, 월로 한다.
  - ex) `2307`
- 디렉토리명은 연, 월, 일로 하고 넘어가는 연, 월에 따라서 구분해서 작성한다.
  - ex) `230710~14`
  - ex2) `230731~0804`
- 파일명은 일, 요일로 작성한다.
  - ex) `10(월).md`
